# ウェル活マスター Pro - 設計図

## 概要

ウェル活（ウエルシア薬局の毎月20日のお客様感謝デー）に特化した、予算管理と在庫チェックを効率化する夫婦向けアプリ。リアルタイム同期により、別々の場所にいてもスムーズな買い物をサポートします。洗練された直感的なUIで、賢い買い物を体験できます。

## デザイン

- **全体的な美学**: モダンで洗練された、プレミアムなユーザーエクスペリエンス。視覚的にバランスの取れたレイアウト、きれいなスペーシング、そして直感的な操作性を重視しています。
- **カラーパレット**: 落ち着いた印象を与える`slate`カラーをベースに、ウェルシアのブランドカラーであるエネルギッシュな赤 (`#e60012`) をアクセントとして使用。ヘッダーやボタンにはグラデーションを適用し、視覚的な魅力を高めています。
- **タイポグラフィ**: 読みやすく美しい「Noto Sans JP」フォントをGoogle Fontsから導入。情報の階層を明確にするため、ヒーローテキスト、ヘッドライン、本文などでフォントサイズと太さを使い分けています。
- **UIコンポーネント**:
    - **カード (`BudgetCard`, `ShoppingList` アイテム)**: 柔らかな深いドロップシャドウ (`shadow-xl`) を使用し、コンポーネントが背景から浮き上がって見えるような「リフト感」を演出。
    - **ボタン**: フォーカス時に「グロー」効果を持たせたインタラクティブなデザイン。丸みを帯びた形状で、ユーザーのアクションを促します。
    - **フォーム**: `@tailwindcss/forms` プラグインを利用し、モダンで使いやすい入力フィールドを実装。
- **アイコン**: `lucide-react` のモダンなアイコンが、ユーザーの理解を助け、ナビゲーションを容易にします。
- **視覚効果**: 多層のドロップシャドウで奥行きを表現。メインの背景には subtle なテクスチャ (`bg-slate-50`) を適用し、触感の良いプレミアムな印象を与えています。

## 機能

### 主要機能

1.  **リアルタイム同期 (Firestore)**
    - 買い物リストの状態をデバイス間でリアルタイムに同期。チェック状況が即座に反映されます。
2.  **予算管理**
    - `BudgetCard` にて、設定した予算と使用済み金額、残金を常に確認可能。
    - グラデーション付きのプログレスバーで、予算の使用率を視覚的に把握できます。
3.  **買い物リスト管理**
    - 商品の追加、削除、および購入済み（カゴ入れ）のチェックが可能。
    - 購入済みの商品はスタイル（打ち消し線、色の変更）が変わり、リストの視認性を向上。
    - リストが空の場合は、ユーザーをガイドする親切なメッセージを表示。
4.  **レスポンシブデザイン**
    - スマートフォンでの利用に最適化されたレイアウト。

## 技術スタック

- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Forms: @tailwindcss/forms
- Icons: Lucide React
- Database: Firebase (Firestore)
- State Management: SWR (for real-time data fetching)

## 今回の実装内容：デザインの全面改修とUI/UXの向上

1.  **デザインシステムの刷新**:
    - `tailwind.config.ts` に `slate` カラーパレットと `@tailwindcss/forms` プラグインを追加。
    - `app/layout.tsx` で、フォントを `Inter` から `Noto Sans JP` に変更し、全体の背景色とテキストカラーを調整。
2.  **ヘッダーとフッターの改善**:
    - `app/page.tsx` のヘッダーにグラデーション背景を適用し、タイトルフォントを太く、大きく変更。
    - フッターの「商品を追加」ボタンを、より大きく、インタラクティブなデザインに更新。
3.  **`BudgetCard` のリファイン**:
    - カードに立体的なドロップシャドウ (`shadow-xl`) と上部のアクセントボーダーを追加。
    - 残金額のフォントサイズを大きくし、グラデーションテキストを適用。
    - プログレスバーにグラデーションとシャドウを追加し、よりモダンな外観に。
4.  **`ShoppingList` の視認性向上**:
    - リストアイテムをシャドウ付きのカードデザインに変更。
    - チェックボックスを大きくし、チェック時のスタイル（色、アイコンのシャドウ）を明確化。
    - リストが空の場合の表示を、アイコン付きでより親切なメッセージに変更。
5.  **`AddItemModal` のモダン化**:
    - モーダルのフォーム要素 (`input`) を `@tailwindcss/forms` を利用してスタイリング。
    - ボタンのデザインを他のコンポーネントと統一し、全体的な一貫性を確保。
