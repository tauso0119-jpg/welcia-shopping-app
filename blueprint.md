# 星家在庫管理 - 設計図

## 概要

星家の在庫と買い物をスマートに管理するための専用アプリ。リアルタイム同期により、家族間でのスムーズな情報共有を実現し、買い物の重複や買い忘れを防ぎます。洗練された直感的なUIで、賢い在庫管理をサポートします。

## デザイン

- **全体的な美学**: モダンで洗練された、プレミアムなユーザーエクスペリエンス。視覚的にバランスの取れたレイアウト、きれいなスペーシング、そして直感的な操作性を重視しています。
- **カラーパレット**: 落ち着いた印象を与える`slate`カラーをベースに、エネルギッシュな赤 (`#e60012`) をアクセントとして使用。ヘッダーやボタンにはグラデーションを適用し、視覚的な魅力を高めています。
- **タイポグラフィ**: 読みやすく美しい「Noto Sans JP」フォントをGoogle Fontsから導入。情報の階層を明確にするため、ヒーローテキスト、ヘッドライン、本文などでフォントサイズと太さを使い分けています。
- **UIコンポーネント**:
    - **カード (`BudgetCard`, `ShoppingList` アイテム)**: 柔らかな深いドロップシャドウ (`shadow-xl`) を使用し、コンポーネントが背景から浮き上がって見えるような「リフト感」を演出。
    - **ボタン**: フォーカス時に「グロー」効果を持たせたインタラクティブなデザイン。丸みを帯びた形状で、ユーザーのアクションを促します。
    - **フォーム**: `@tailwindcss/forms` プラグインを利用し、モダンで使いやすい入力フィールドを実装。
- **アイコン**: `lucide-react` のモダンなアイコンが、ユーザーの理解を助け、ナビゲーションを容易にします。
- **視覚効果**: 多層のドロップシャドウで奥行きを表現。メインの背景には subtle なテクスチャ (`bg-slate-50`) を適用し、触感の良いプレミアムな印象を与えています。

## 機能

### 主要機能

1.  **リアルタイム同期 (Firestore)**
    - 買い物リストの状態をデバイス間でリアルタイムに同期。チェック状況が即座に反映されます。
2.  **予算管理**
    - `BudgetCard` にて、設定した予算と使用済み金額、残金を常に確認可能。
    - グラデーション付きのプログレスバーで、予算の使用率を視覚的に把握できます。
3.  **買い物リスト管理**
    - 商品の追加、削除、および購入済み（カゴ入れ）のチェックが可能。
    - 購入済みの商品はスタイル（打ち消し線、色の変更）が変わり、リストの視認性を向上。
    - リストが空の場合は、ユーザーをガイドする親切なメッセージを表示。
4.  **レスポンシブデザイン**
    - スマートフォンでの利用に最適化されたレイアウト。

## 技術スタック

- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Forms: @tailwindcss/forms
- Icons: Lucide React
- Database: Firebase (Firestore)
- State Management: SWR (for real-time data fetching)
